# Simple E-Commerce Website Demo

A beginner-friendly e-commerce website built with HTML, CSS, and JavaScript featuring different product categories.

## Features

- Product catalog with multiple categories: Electronics, Clothing, Footwear, Kids
- Clickable "Buy Now" buttons that show purchase confirmations
- Responsive design
- Simple navigation between Home, Products, Login, and Sign Up pages

## How to Use

1. Open `index.html` in your web browser to view the home page.
2. Browse products arranged by categories in order: Electronics (gadgets), Clothing, Footwear, Kids.
3. Visit the Products page for all items in a single view.
4. Click the "Buy Now" buttons to see demo purchase alerts.
5. Use Login and Sign Up pages for authentication demos.

## Files

- `index.html` - Home page with categorized product sections
- `products.html` - Full product catalog
- `login.html` - Login page
- `signup.html` - Sign up page
- `styles.css` - Styling
- `script.js` - JavaScript for button interactions and product display

## Product Categories

- **Electronics**: Headphones, Smart Watch, Laptop, Joystick, Gaming Mouse, Bluetooth Speaker
- **Clothing**: T-Shirts, Jeans, Winter Jacket
- **Footwear**: Running Shoes, Casual Sneakers, Formal Shoes
- **Kids**: Toy Car, Children's Book, Kids Backpack

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Browser Support

Works in all modern browsers.

## Note

This is a simple demo for beginners to learn HTML, CSS, and JavaScript. The buttons are just for demonstration and show alerts with purchase confirmations.
